// 使用方法

标准 bin.gz 权重（从心 / 智子速降LR / 凤鸣）

直接配合官方 KataGo 使用：

./katago gtp -model congxin-260217.bin.gz -config gtp_example.cfg

Transformer ONNX 权重（b18tfrs · 首推）

需要使用 KataGomo fork 的 onnx-general 分支编译的引擎：

1. 克隆并编译
git clone -b onnx-general https://github.com/hzyhhzy/KataGomo_fork.git
cd KataGomo_fork/cpp
cmake . -DUSE_BACKEND=TENSORRT -DBUILD_DISTRIBUTED=0
make -j 4
2. 运行
./katago gtp -model b18c384h12tfrs_1_fd2-s3268823040-d1410226031.onnx -config gtp_example.cfg
